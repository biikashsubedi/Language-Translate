## INSTALLATION

pip3 install translator


## RUN

python3 type_translate.py [default JAPANESE]<br>


language option:<br>
'en' = english<br>
'ja' = japanese<br>
'ne' = nepalese<br>
'ge' = gernmany<br>
'es' = espanish<br>
'hi' = hindi<br>

[NOTE: Search on wikipedia for more language shortcut]<br>
